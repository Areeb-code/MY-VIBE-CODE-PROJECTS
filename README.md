# ⚡ Vibe Coded Projects

This repository is a collection of applications built using **vibe coding** — a fast, intuitive development approach where ideas are rapidly transformed into working prototypes using AI assistance and creative iteration.

---

## 🧠 What is Vibe Coding?

Vibe coding is a development mindset focused on:

- ⚡ Speed over perfection
- 🎯 Idea-to-execution in minimal time
- 🤖 Leveraging AI tools for building
- 🔁 Iterative experimentation
- 🧪 Learning by building real things

Instead of over-planning, the goal is to **build first, refine later**.
